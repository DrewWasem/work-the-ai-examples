# PDF #01 — Claude Code as Your Primary Engineer

Companion files for the PDF. Get the PDF at [work-the-ai.com](https://work-the-ai.com) for the patterns these files plug into.

## What's here

- **[`claude-md-starter/CLAUDE.md`](./claude-md-starter/CLAUDE.md)** — drop at the root of any project. Fill in the blanks (or ask Claude to fill them in for you). Your next session starts informed.
- **[`prompt-spec/5-part-skeleton.md`](./prompt-spec/5-part-skeleton.md)** — the 5-part prompt structure as a copy-paste template. Fill in the blanks for every spec you hand the agent.
- **[`auto-commit-hook/`](./auto-commit-hook/)** — a drop-in `.claude/settings.json` Stop hook that auto-commits after every Claude Code session. Your "undo button."

## What's NOT here

The PDF's prose, examples, walkthroughs, and case study. Those are the paid product. This repo is the runnable scaffolding only.
